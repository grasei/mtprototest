# Telegram Proxy Checker 

**Инструмент для проверки MTProto-прокси и автоматического поиска новых прокси в Telegram-каналах**

---
**Важно**  
Сделано для себя. Протестировано только под Windows 11.  Никакой официальной поддержки.
---

## 📌 Описание

 Автоматический поиск MTProto-ссылок на прокси в Telegram-каналах с проверкой на работоспособность

Основан на коде [Proxy-Checker](https://github.com/exelite-dev/Proxy-Checker) от exelite-dev (устанавливается отдельно при желании).

**Важно:** Для работы используются API-ключи от официального приложения Telegram Desktop (API_ID и API_HASH).

---

## ✨ Особенности


- Поддержка только **MTProto** прокси (стандарт Telegram)
- Асинхронная проверка найденных прокси на работоспособность
- Извлечение прокси из текста сообщений и инлайн-кнопок
- Автоматическая проверка найденных прокси на работоспособность
- Генерация кликабельной HTML-страницы для быстрого подключения
- Автоматическое удаление дубликатов и учёт сбоев
- Совместимость с Windows (подавление ошибок соединения)
- Может в качестве  использовать из прокси-сервера working.txt (если существует) от [Proxy-Checker](https://github.com/exelite-dev/Proxy-Checker) (путь прописывается в settings.py)




## 📦 Требования

- Python 3.10+
- Telethon
- qrcode (для QR-входа)
- aiohttp
- rich

## 🔧 Конфигурация

Отредактируйте файл `settings.py`:

```python
# Telegram-каналы для мониторинга (вы должны быть подписаны на них)
# Это вымышленные примеры. Добавьте свои.
CHANNELS = ["mtp4", "TPRU", "ProxyMTP"]


# Используйте ключи от официального приложения Telegram Desktop

API_ID = 2040
API_HASH = 'ваш_api_hash_здесь'  # (или API_HASH = 'b18441a1ff607e10a989891a5462e627')

# Файлы ввода/вывода
CONFIG_FILE = "config.txt"              # Стартовые прокси
WORKING_FILE = "Proxy-Checker-main/working.txt"  # Дополнительный источник прокси
HTML_FILE = "proxies.html"              # Выходная HTML-страница
```



## 🎯 Использование


python telegram_channel_checker.py


**Что делает:**
1. Загружает прокси из `config.txt` и `working.txt`
2. Подключается к Telegram (**при первом запуске - вход по QR-коду**)
3. Сканирует настроенные каналы на наличие ссылок на прокси
4. Параллельно проверяет найденные прокси (макс. 5 одновременно)
5. Сохраняет рабочие прокси в `config.txt`
6. Генерирует `proxies.html` с кликабельными ссылками
7. Завершает работу и жмёт нажатия клавиши

## 🔐 Безопасность и конфиденциальность

- Никакие данные прокси не передаются на внешние сервера
- Вся проверка прокси выполняется локально
- Файл сессии Telegram (`my_account.session`) хранится локально
- Ваши API-ключи никогда не передаются

---

## ⚠️ Важные замечания

- **Вы должны быть подписаны на мониторимые Telegram-каналы**
- **При первом запуске требуется авторизация по QR-коду или телефону**
- На Windows ошибки сброса соединения автоматически подавляются
- Channel Scraper работает в один проход (нет бесконечного цикла)
- Файл `working.txt` доступен только для чтения и не изменяется
- **API-ключи идентичны ключам официального Telegram Desktop**
